## 模块

```
- file                        # 文件上传
```

## 目录结构

```
├── logs                      # 日志
├── public                    # 静态文件
├── src                       #
│   ├── routes                # 路由
│   └── app.js                # 入口
├── README.md
├── networks.json             # 项目所需配置
└── package.json
```

## 使用

```bash
$ npm install                 # 安装项目依赖
$ npm start                   # 运行项目
$ npm run dev                 # 开发环境运行
$ npm run prd                 # 生产环境运行
```
